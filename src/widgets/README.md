# Widget Components

## Overview
Reusable UI components for application layout and navigation.

## Components

### Header
- **Location**: `src/widgets/Header/`
- **Components**: Header, ProfileMenuAvatar, NotificationPanel
- **Features**: User profile, notifications, search, branding

### Sidebar
- **Location**: `src/widgets/Sidebar/`
- **Features**: Navigation menu, responsive collapse, route highlighting

### Footer
- **Location**: `src/widgets/Footer/`
- **Components**: Footer, MobileBottomNav
- **Features**: Links, copyright, mobile navigation

## Usage
```tsx
import { Header } from '@/widgets/Header';
import { Sidebar } from '@/widgets/Sidebar';
import { Footer } from '@/widgets/Footer';
```

## Features
- Responsive design
- Theme integration
- Accessibility compliant
- TypeScript support
- MUI component integration
